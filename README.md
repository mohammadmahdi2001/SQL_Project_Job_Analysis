# SQL Job Market Analysis

## Project Overview

This project analyzes data analyst job postings using SQL. The goal is to identify top-paying jobs, in-demand skills, and the most valuable skills to learn based on salary and market demand.

## Questions Answered

1. What are the top-paying data analyst jobs?
2. What skills are required for top-paying jobs?
3. What skills are most in demand?
4. Which skills are associated with higher salaries?
5. What are the most optimal skills to learn?

## Tools Used

- SQL
- PostgreSQL
- Visual Studio Code
- Git & GitHub

## Key Findings

- SQL is the most demanded skill for data analysts.
- Python, Tableau, and Excel are among the most requested skills.
- Cloud technologies such as AWS, Azure, and Snowflake are linked to higher salaries.
- Remote data analyst positions can offer very competitive salaries.

## Skills Demonstrated

- SQL Queries
- Joins
- Common Table Expressions (CTEs)
- Aggregate Functions
- Data Analysis
- Data Filtering and Sorting

## Conclusion

This project helped me strengthen my SQL skills and gain practical experience in analyzing real-world job market data. The results provide useful insights into the skills and technologies that are valuable in the data analytics field.
